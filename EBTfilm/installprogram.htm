Комплект файлов для установки программы

Программа поставляется в виде архива EBTfilm.zip

 

Содержимое архива:

•root_v5.34.32.win32.vc10.exe - установочный пакет системы ROOT (http://root.cern.ch)

•AnalyzeMacros - директория с ROOT макросами предназначенными для анализа данных.

•CalibratonMacros - директория с ROOT макросами предназначенными для калибровки дозиметрических плёнок.

•Setup.bat - установочный файл

•Uninstall.bat - файл удаления программы

•Tif2RootReg.bat - файл устанавливающий ассоциацию файлов с расширением .tiff с запуском программы анализа данных

•RemoveTif.reg - файл удаляющий ассоциацию файлов с расширением .tiff с запуском программы анализа данных

•vsvars10.sh, .bash_profile - файлы необходимые для разработчиков
